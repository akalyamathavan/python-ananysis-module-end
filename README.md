# python-ananysis-module-end
Social Media Engagement Analytics

 Project Overview

This project analyzes a social media engagement dataset using Python.
The goal is to understand user behavior, content performance, and engagement patterns.
Data analysis and visualization techniques were applied to convert raw data into meaningful insights.

Dataset

The dataset contains information about users and posts, including:

- Age, Gender, Country
- Post Type & Category
- Likes, Comments, Shares
- Impressions & Watch Time
- Follower Count
- Sentiment
- Device Type

 Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

 Project Workflow

 Data Loading

- Imported CSV using Pandas
- Checked data types
- Converted date columns to datetime

 Data Cleaning

- Handled missing values
- Removed duplicates
- Standardized categories
- Extracted hashtag count
- Cleaned sentiment labels

 Data Exploration

- Used "head()", "info()", "describe()"
- Analyzed categorical distributions
- Created correlation matrix
- Generated groupby summaries

 Data Wrangling

- Created new fields:
   - Engagement Score
   - Hashtag Count
- Performed grouped summaries by:
   - Post Type
   - Country
   - Sentiment
  Statistical Analysis

Calculated descriptive statistics for engagement metrics:

- Mean, Median, Mode
- Standard Deviation & Variance
- Percentiles
- Skewness

 Data Visualization

Created multiple visualizations using:

- Scatter Plot
- Count Plot
- Bar Plot
- Violin Plot
- Pair Plot
- Heatmap
- Swarm Plot
- Interactive Plotly Chart
  
 Key Insights

- Engagement varies across post types and categories
- Posting time affects impressions
- Device type impacts watch time
- Verified accounts show different performance
- Negative sentiment posts received highest engagement
- Negative posts gained more likes and shares, while neutral posts generated more comments

 Conclusion

This project demonstrates how data analysis techniques help uncover patterns in social media engagement.
The findings provide useful insights for improving content strategy and audience targeting.
